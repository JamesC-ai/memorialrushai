# MemorialRushAI

MemorialRushAI is a privacy-first rush memorial video intake and planning tool.

MVP:

- Browser-local intake form.
- Tribute video scene plan.
- Caption and memory text draft.
- Editor handoff brief.
- Copy and email order pack.
- Privacy, terms, support, sitemap, robots, and SEO pages.

Commercial path:

- Free planner.
- $49 24-hour tribute video starter.
- $99+ expanded tribute video.
- Human final review required for names, dates, photo order, tone, and music rights.
